# Prince Patel Academic Website

GitHub Pages repository: `PRINCE0856.github.io`

## Upload
Upload every file and the `assets` folder directly to the root of the repository. `index.html` must remain at the repository root.

## Publish
In GitHub: **Settings → Pages → Deploy from a branch → main → /(root) → Save**.

The website will appear at: `https://PRINCE0856.github.io`
